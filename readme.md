# 智能個人財務管理系統 (Smart Finance Manager)

這是一個基於 **Python** 開發的桌面端個人財務管理應用程式。旨在幫助使用者追蹤日常收支、設定財務目標，並透過視覺化圖表與自動化建議優化個人的理財習慣。

---

### 1. 登入與權限管理
系統支援多使用者註冊與登入，確保個人財務資料的安全與隱私。

## 🚀 核心功能

* **使用者驗證系統**：支援多使用者帳號切換，保護資料獨立性。
* **收支紀錄管理**：提供類別篩選（如：餐飲、薪資、交通等）與金額存取。
* **歷史明細查詢**：以表格形式（Treeview）清晰呈現所有交易流水。
* **視覺化圖表分析**：動態生成支出結構比例圖，掌握資金去向。
* **財務建議與預算追蹤**：設定預算上限，系統自動進行超支警示與健檢分析。
* **本地資料庫存取**：使用 `SQLite` 進行資料持久化儲存，確保資料不遺失。

---

## 🏗️ 系統架構

本專案採用 **MVC (Model-View-Controller)** 的設計思想開發：

* **Model (資料層)**：`DBManager` 類別，負責 SQLite 資料庫的 CRUD 操作。
* **View / Controller (介面與邏輯層)**：`FinanceApp` 類別，負責 Tkinter GUI 渲染與業務邏輯處理。

---

## 🛠️ 技術棧 (Tech Stack)

* **程式語言**：Python 3.x
* **圖形介面**：Tkinter (Python Standard Library)
* **資料庫**：SQLite 3
* **數據繪圖**：Matplotlib
